# TECHIT BANGLADESH – Website

A premium, modern, and responsive landing page for TECHIT BANGLADESH.  
Includes sleek hero section, animated background, and all social links with glassmorphic card UI.

## Features
- Glassmorphism and smooth gradient accent
- Animated hero background (particles + waves)
- Responsive: mobile, tablet, desktop
- Premium elegant design, no clutter
- All official social links in elegant card grid
- Minimal, SEO-optimized, fast loading
- Semantic HTML & modern accessibility best-practices
- Icons for all major social platforms, custom SVG where needed

## Quick Start

```bash
npm install
npm start
```
or

```bash
yarn install
yarn start
```

## Tech Stack

- React (+ TypeScript)
- Styled Components
- react-icons

## Production Build

```bash
npm run build
```
or

```bash
yarn build
```

---

## Customizing

- Update logo, icons, colors, or links in `src/components/SocialLinksSection.tsx`
- To change background/animation: edit `HeroSection.tsx`
